# 🧘 ZenMat – AI Powered Smart Yoga Mat

ZenMat is an **AI-powered smart yoga mat** designed to help users improve their yoga practice through **real-time posture correction, sensor-based body tracking, and personalized guidance**.

The system integrates **Artificial Intelligence, IoT sensors, and network communication** to analyze body posture and provide instant feedback.

---

# 🚀 Features

- **Real-Time Posture Detection**
  - Uses a camera and AI model to detect yoga poses and identify incorrect posture.

- **Sensor-Based Position Tracking**
  - FSR (Force Sensitive Resistor) sensors embedded in the mat detect body pressure and positioning.

- **AI-Based Feedback System**
  - Machine learning model evaluates posture accuracy and provides **real-time visual and audio feedback**.

- **Web Dashboard**
  - A **Flask-based web application** allows users to:
  - View posture analysis
  - Track yoga sessions
  - Monitor improvement over time

- **Session History**
  - Stores previous sessions to help users track progress and performance.

---

# 🌐 Networking & System Communication

ZenMat demonstrates **practical networking concepts through IoT device communication and client–server architecture**.

### System Communication Flow

1. **IoT Sensors → Microcontroller**
   - FSR sensors detect pressure and body position on the mat.

2. **Microcontroller → Backend Server**
   - Sensor data is transmitted to the backend server over the network.

3. **Camera → AI Processing Module**
   - Camera captures user movements and sends frames for AI posture analysis.

4. **Backend Server → Web Dashboard**
   - Processed data is delivered to the web application through **Flask REST APIs**.

### Networking Concepts Used

- Client–Server Architecture  
- REST API Communication  
- IoT Device Data Transmission  
- Real-Time Data Processing  
- Hardware–Software Integration
